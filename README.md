# Présentation

Ce projet consiste à développer une application permettant au service **DRH** de gérer les congés et les absences des employés.

Le système centralise les informations liées aux employés et facilite le suivi des demandes de congé.

Le système peut aussi calculer les soldes de congé et garder un historique des absences.

---

# Acteurs du système

## DRH

Utilisateur principal du système.

Responsabilités

- gérer les employés
- enregistrer les demandes de congé
- valider ou refuser les congés
- consulter les informations des employés

## Système

Rôle automatique

- calculer les soldes de congé
- vérifier les conflits de dates
- générer des alertes ou notifications

# Fonctionnalités principales

## 1 Gestion des employés

Le DRH peut

- ajouter un employé
- modifier les informations d'un employé
- désactiver un employé
- consulter la liste des employés
- rechercher un employé

---

## 2 Gestion des congés

Le DRH peut

- enregistrer une demande de congé
- consulter les demandes de congé
- valider une demande
- refuser une demande
- consulter le solde de congé d'un employé

Le système peut

- vérifier les conflits de dates
- mettre à jour automatiquement les soldes

---

## 3 Historique des congés

Le système garde

- les congés pris
- les congés validés
- les congés refusés

Cela permet au DRH de suivre les absences des employés.
