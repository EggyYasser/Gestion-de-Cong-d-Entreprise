## Présentation

Ce projet a pour but de développer une application de **gestion des congés d'entreprise**.  
Elle permet de faciliter l'organisation des demandes de congé entre les employés et l'administration.

L'objectif principal est de remplacer la gestion manuelle par une solution simple, claire et efficace.

---

## Objectifs du projet

- Gérer les employés
- Gérer les demandes de congé
- Permettre la validation ou le refus des demandes
- Suivre l'historique des congés
- Simplifier le travail administratif
- Améliorer l'organisation interne de l'entreprise

---

## Fonctionnalités prévues

### Partie Employé
- Consulter son solde de congés
- Envoyer une demande de congé
- Voir l'état de la demande
- Consulter l'historique de ses congés

### Partie Administration
- Ajouter, modifier et supprimer des employés
- Consulter les demandes de congé
- Accepter ou refuser une demande
- Gérer les types de congés
- Suivre les statistiques globales
